# ClickFunnels-WP

Import a clickfunnel page inside Wordpress without changing your domain DNSs

## Getting Started

1 - Install as any wp plugin<br>
2 - Use the shortcode [clickfunnels url=""]. THe url must be filled with the clickfunnels page url.<br>
